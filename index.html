About

As a complete decentralized(crypto) world , EMIT crypto technology is based on three different levels of continuously evolving information systems built from “Electronic Material Information Technology”.

In the decentralized(crypto) world of EMIT, economic structures and cycles transcend those of the real world.

Systems
In EMIT’s architecture, there are three major systems, namely “EMIT Core Protocol”, “EPOCH Ecosystem” and “GDP Financial Market”, which extend from the basic bottom to the upper layer.
EMIT Core Protocol
The EMIT protocol network is a consensus network of decentralized nodes, which is the infrastructure that supports the operation of the entire EMIT system.
EPOCH Ecosystem
EPOCH ecosystem refers to various decentralized(crypto) ecologies running on the EMIT network based on the EPOCH protocol and framework for building the endogenous cycle of DWE.
GDP Financial Market
GDP Financial Market is one of the most important core applications in EPOCH and a decentralized trading and auction center for diversified assets, supporting diverse assets and trading behavior.

Projects
EMIT Wallet

EMIT Wallet is a client application that runs on PC browsers and Android / IOS mobile phones and can be connected to any EMIT nodes. With it, you can:

  • Manage the assets in your EMIT account
  • Manage assets transferred from other blockchains
  • Participation in various EPOCH ecosystems
  • Develop Dapps for the EPOCH ecosystem
  • Use GDP Financial Market
EMIT Node

An EMIT full node program which can automatically joins the EMIT P2P network. The EMIT wallet can be connected to any EMIT nodes.
EMIT Negentropy Generator(Coming soon)

A client program that connects to the EPOCH Dapp and provides hashrate to Epoch. Hashrate exist as a data called Negentropy. Negentropy is the original driving force behind EPOCH.
EMIT Browser(Coming soon)

The browser of EMIT DW, users can view the operation of the whole EMIT system through it.
